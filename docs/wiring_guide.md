# TB6612FNG ↔ Raspberry Pi 5 接線指南

## 接線總覽

```
 18650 x2 (7.4V)          Raspberry Pi 5 (5V 行動電源供電)
      │                         │
      ▼                         ▼
   ┌──────────────────────────────────────┐
   │            TB6612FNG                 │
   │                                      │
   │  VM ←── 7.4V (電池正極)              │
   │  VCC ←── 3.3V (RPi Pin 1)           │
   │  GND ←── 共地 (電池GND + RPi GND)   │
   │                                      │
   │  STBY ←── GPIO24 (Pin 18)           │
   │                                      │
   │  AIN1 ←── GPIO17 (Pin 11)  ──┐      │
   │  AIN2 ←── GPIO27 (Pin 13)    ├─ 左馬達
   │  PWMA ←── GPIO12 (Pin 32)  ──┘      │
   │  AO1 ──→ 左馬達 +                   │
   │  AO2 ──→ 左馬達 -                   │
   │                                      │
   │  BIN1 ←── GPIO22 (Pin 15)  ──┐      │
   │  BIN2 ←── GPIO23 (Pin 16)    ├─ 右馬達
   │  PWMB ←── GPIO13 (Pin 33)  ──┘      │
   │  BO1 ──→ 右馬達 +                   │
   │  BO2 ──→ 右馬達 -                   │
   └──────────────────────────────────────┘
```

## 詳細接線表

| TB6612FNG 腳位 | 連接至 | 說明 |
|:---:|:---:|:---|
| **VM** | 18650 電池正極 (7.4V) | 馬達電源輸入 |
| **VCC** | RPi 3.3V (Pin 1) | 邏輯電源 |
| **GND** | RPi GND (Pin 6) + 電池負極 | ⚠️ **共地非常重要！** |
| **STBY** | GPIO24 (Pin 18) | 拉高=啟用，拉低=待機 |
| **AIN1** | GPIO17 (Pin 11) | 左馬達方向控制 1 |
| **AIN2** | GPIO27 (Pin 13) | 左馬達方向控制 2 |
| **PWMA** | GPIO12 (Pin 32) | 左馬達速度 (PWM) |
| **AO1** | 左馬達 + | 馬達 A 輸出 1 |
| **AO2** | 左馬達 − | 馬達 A 輸出 2 |
| **BIN1** | GPIO22 (Pin 15) | 右馬達方向控制 1 |
| **BIN2** | GPIO23 (Pin 16) | 右馬達方向控制 2 |
| **PWMB** | GPIO13 (Pin 33) | 右馬達速度 (PWM) |
| **BO1** | 右馬達 + | 馬達 B 輸出 1 |
| **BO2** | 右馬達 − | 馬達 B 輸出 2 |

## 馬達方向控制真值表

| IN1 | IN2 | 動作 |
|:---:|:---:|:---:|
| HIGH | LOW | 正轉 |
| LOW | HIGH | 反轉 |
| LOW | LOW | 停止（滑行） |
| HIGH | HIGH | 剎車 |

## ⚠️ 注意事項

1. **共地**：Raspberry Pi GND 與 18650 電池 GND **必須連接在一起**，否則訊號無法正確傳遞。
2. **切勿將 7.4V 連到 VCC**：VCC 只能接 3.3V 或 5V 邏輯電源，VM 才是馬達的高壓電源輸入。
3. **GPIO12 和 GPIO13** 是 Raspberry Pi 的硬體 PWM 腳位，能提供更穩定的 PWM 控制。
4. **先測試馬達方向**：如果馬達轉的方向相反，交換該馬達的 AO1/AO2（或 BO1/BO2）接線即可。
5. **USB 攝影機**直接插入 Raspberry Pi 的 USB 埠即可，建議使用 USB 3.0 埠以獲得更穩定的資料傳輸。
